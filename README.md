---

## Overview

This project uses **Machine Learning (Logistic Regression)** to classify whether a person is likely to have diabetes based on health metrics such as glucose level, BMI, insulin, and more.  
The frontend is built using **Flask templates (HTML, CSS)** with a stylish **purple glassmorphism theme** for a clean, modern look.

---

## Features

- Predicts diabetes using a trained Logistic Regression model
- Preprocessing with `StandardScaler` for consistent input scaling
- Attractive UI with gradient purple background and glass effect
- Simple and user-friendly form to input patient details
- Fully integrated with Flask backend for live predictions
